How this dashboard works

This UI is powered by a FastAPI / NodeJS microservices backend running in Kubernetes and exposed via a REST API.

The page calls /api/orders on this service, which proxies to the internal order-service endpoint in the cluster (ORDER_SERVICE_URL environment variable).
